📖 使い方
  1. game.html を選択
  2. 遊ぶ人に伝えるパスワードを入力（2回）
  3. ボタンをクリック → 暗号化された index.html がダウンロードされる
  4. ダウンロードした index.html を penguin-highway フォルダの index.html に上書きコピー
  5. ターミナルで cd ~/penguin-highway && git add index.html && git commit -m "Update encrypted build" && git push
  6. 1〜2分後、公開ページがパスワード画面になる

⚙️ AES-256-GCM + PBKDF2 (250,000 iterations, SHA-256) — Web Crypto API を使用
